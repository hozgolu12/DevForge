# DevForge Prompt Pack

This pack contains scaffold prompts for building DevForge with an AI coding agent.

Use each prompt sequentially after validating the previous phase.

Phases:
1. Foundation
2. Node.js Platform
3. Python Platform
4. AI Platform
5. Java Platform
6. Flutter Platform
7. Databases
8. Infrastructure & Monitoring
9. DevForge CLI
10. Production Hardening

General instructions for every phase:
- Work in the current repository.
- Never recreate completed work.
- Validate Docker Compose after changes.
- Run formatting/linting where applicable.
- Prefer production-grade defaults.
- Avoid TODOs and placeholder code.
